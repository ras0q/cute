# Cute

Cute: A CLI tool to exe"CUTE"s commands from markdown files.

## Usage

```sh
$ source ./cute.sh
$ cute -h
Cute: A CLI tool to exe"CUTE"s commands from markdown files.

Usage:
  cute [-h] [-d]

Options:
  -h: Show this help message and exit
  -d: Enable debug mode (prints commands as they are executed)

Example:
  cute -d
This will enable debug mode.
```
